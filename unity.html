> Elders Demise

About

Elders Demise was an ambitious project I undertook as part of my bachelor's program at KdG University of Applied Sciences. Taking inspiration from Arkane Studios' renowned title, Dishonored, I set out to create a game world and story from scratch.

This project served as a personal challenge to assess my capabilities and determine if I had the potential to become a professional game developer. Despite still having limited knowledge in the field at the time.

Throughout the intensive six-month development period, I encountered numerous challenges, learning about them and solving them only kept fueling my growing passion for game development.

Elders Demise is a significant milestone in my journey. It accelerated my growth as a game developer and I hope to one day tackle this project again.

If you'd like to learn more about the project and explore the Game Design Document, please find it [here]. Thank you for your interest!
