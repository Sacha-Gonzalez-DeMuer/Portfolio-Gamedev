> Elders Demise

Download Demo

About

Elders Demise was an ambitious solo project I undertook as part of my bachelor's program at KdG University of Applied Sciences. Taking inspiration from Arkane Studios' renowned title, Dishonored, I set out to create a game world and story from scratch.

The demo offers players the flexibility to complete the level in multiple ways: they can choose an aggressive approach, taking a straightforward path to the objective; alternatively, they can opt for a stealthy strategy by infiltrating the enemy outpost; or they can find escapes by exploring the environment using parkour to navigate their way through.

Elders Demise is a significant milestone in my journey. It accelerated my growth as a game developer and I hope to one day tackle this project again. If you'd like to learn more about the project and explore the Game Design Document, please find it [here]. Thank you for your interest!

> Buddy Jumping

GitHub Download Demo

About

Embarking on a research project as part of the Gameplay Programming course at HowestDAE, I sought to unlock the mysteries of player AI engines. While chess engines fascinate me, I decided to take a different approach and explore the implementation of the Monte Carlo Tree Search (MCTS) algorithm for the game of Connect 4. I investigated strategies for optimizing and improving the model and learned a great deal about the intricacies of adaptive search algorithms and decision-making processes in game-playing AI.

Read about my research on the ReadMe page.
