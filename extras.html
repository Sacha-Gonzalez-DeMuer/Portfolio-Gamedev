About

Using WPF and XAML, I set out to create a hero info screen for Valorant. During this exercise, I learned to parse JSONs and make use of online APIs, as well as asynchronous programming in C#.

To enhance the structure and maintainability of the project, I implemented the MVVM architectural pattern.


• Applied MVVM architectural pattern
• JSON parsing
• Usage of online APIs
